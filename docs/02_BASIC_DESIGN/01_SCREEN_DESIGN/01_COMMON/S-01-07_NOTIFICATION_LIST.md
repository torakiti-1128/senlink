# S-01-07 通知一覧

## 1. 概要
- 対象ロール：ALL（ログイン済み前提）
- 目的：自分宛の通知を一覧で確認し、未読／既読の管理を行う。
- 遷移元：
  - ヘッダーの通知アイコン（共通部品）
- 遷移先：
  - 通知詳細：S-01-08

## 2. 画面構成
- ヘッダー：あり（ロール別ヘッダー、共通部品定義に従う）
- メイン領域：
  - 一覧（通知センター）
  - フィルタ（未読のみ、種別など）
  - ページング
  - 一括既読ボタン
- フッター：あり

## 3. 表示項目
- 通知一覧（行／カード）
  - タイトル
  - 受信日時（作成日時）
  - 既読／未読
  - 種別（type）
- 一覧上部
  - 未読件数
- エラーメッセージ（概要レベルで可）
  - セッション無効：S-01-02へ
  - 学外アクセス（学内IP制限）：S-01-01へ

## 4. 操作
- 一覧の行選択
  - 押下：S-01-08（通知詳細）へ遷移
- 既読更新（単体）
  - 詳細画面遷移時に自動既読
- フィルタ／ページング
  - 未読のみ表示、種別絞り込み、ページング

## 5. 関連API
- 通知一覧（通知センター）
  - GET /api/v1/notification/notifications
- 未読件数（バッジ用／一覧上部表示用）
  - GET /api/v1/notification/notifications/unread-count
- 既読更新（単体）
  - PATCH /api/v1/notification/notifications/{notification_id}/read

## 6. 補足
- ヘッダーの未読バッジは、未読件数API（/unread-count）を使用する（共通部品定義と整合）。
