# S-01-01 アクセス不可

## 1. 概要
- 対象ロール：ALL
- 目的：学内IP制限により学外からアクセスしたユーザーに対し、利用不可であることと対処方法を案内する。
- 遷移元：学外IPからSenLinkへアクセスした場合（Apacheの制御により403相当）
- 遷移先：
  - 再試行（同一URLへ）

## 2. 画面構成
- ヘッダー：ロゴのみ（リンク無し）※共通部品定義の例外運用
- メイン領域：
  - アクセス不可メッセージ
  - 対処方法の案内
- フッター：あり

## 3. 表示項目
- タイトル：学内ネットワーク外からは利用できません
- 本文：
  - 本サービスは学内IP制限により、学外からのアクセスはブロックされます。
  - 学内Wi-Fi／学内ネットワーク／VPNからアクセスしてください。

## 4. 操作
- 再試行：ページリロード（ブラウザ更新）

## 5. 関連API
- なし

## 6. 補足
- 学外アクセス検知/遮断はApache側の制御を前提とする。
- S-01-01はログイン状態に依存しないため、ユーザーメニューや通知導線は表示しない。
