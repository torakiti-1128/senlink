## 2. 学生向け機能（F-02-XX）

| 機能ID | 機能名 | サブ機能ID | サブ機能名 | 詳細 | 備考 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **F-02-01** | **求人支援** | SF-01 | 求人一覧表示 | 種別、案件／企業名、開催日、締切、タグ、状態、レコメンド有無を表示。 | |

| | | SF-05 | ブックマーク | 求人の「気になる」登録と解除。 | |
| **F-02-02** | **活動支援** | SF-01 | 活動サマリ確認 | 応募数集計、直近イベント、要対応タスク数等の表示。 | |
| | | SF-02 | 応募済求人一覧 | 種別、開催日時、ステータス、進捗指標（ToDo完了率等）の表示。 | |
| | | SF-03 | 選考タスク管理 | 企業別タスク（ToDo）の表示、期限超過の強調、完了更新。 | |
| | | SF-04 | 応募求人検索 | ステータス（参加前/済/すべて）、種別、各種ソート。 | |
| | | SF-05 | 辞退申請 | 応募済求人の辞退申請。教員の承認が必要なフロー。 | 申請中は強調 |
| **F-02-03** | **申請支援** | SF-01 | 申請種別選択 | 面談予約、書類発行、内定報告から選択。 | |
| | | SF-02 | 申請情報入力 | 種別ごとに異なるフォームへの入力と送信。 | |
| | | SF-03 | 申請履歴確認 | 過去の申請日時、種別、概要、ステータスの一覧。 | |
| | | SF-04 | 状況／FB確認 | ステータス変化（承認/却下）と教員コメントの閲覧。 | |
| **F-02-04** | **学生プロフィール管理** | SF-01 | 基本情報参照 | 氏名、学籍番号、所属、生年月日、性別、メールの確認。 | 編集不可項目あり |
| | | SF-02 | 就活プロフ編集 | 希望職種、希望勤務地、自己PR、資格、ポートフォリオURL。 | |
| | | SF-03 | 活動状況設定 | 「活動中/活動終了」のステータス切り替え。 | |
| | | SF-04 | システム利用設定 | パスワード変更、ログアウト。 | |
