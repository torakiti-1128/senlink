# 画面一覧および画面遷移図：SenLink（センリンク）

## 1. 画面一覧

### 1.1 共通画面（S-01-XX）
| 画面ID | 画面名 | 説明 |
| :--- | :--- | :--- |
| S-01-01 | ログイン画面 | メールアドレスとパスワード（PW）でログイン |
| S-01-02 | ユーザー登録画面 | メールアドレス入力→ワンタイムパスワード（OTP）送信 |
| S-01-03 | OTP入力画面 | OTP認証 |
| S-01-04 | PW再設定画面 | PWを忘れた時のリセット画面 |
| S-01-05 | 初回設定画面 | ロールに応じた初期プロフィール登録（初回ログイン時のみ） |
| S-01-06 | 通知一覧画面 | 通知一覧（既読／未読管理） |

### 1.2 学生画面（S-02-XX）
| 画面ID | 画面名 | 説明 |
| :--- | :--- | :--- |
| S-02-01 | 求人検索画面 | 求人一覧、フリーワード検索、条件指定、ソート |
| S-02-02 | 求人詳細画面 | 詳細表示、応募／ブックマーク（応募時モーダル表示） |
| S-02-03 | ブックマーク画面 | ブックマークした求人の一覧 |
| S-02-04 | 活動支援画面 | 応募済リスト表示、ToDoタスク管理、就活ダッシュボート |
| S-02-05 | 申請支援画面 | 書類・面談・内定の申請フォームおよび履歴一覧 |
| S-02-06 | 学生マイページ画面 | プロフィール確認／編集、ログアウト |

### 1.3 教員画面（S-03-XX）
| 画面ID | 画面名 | 説明 |
| :--- | :--- | :--- |
| S-03-01 | 教員ダッシュボード画面 | クラス全体の進捗サマリ、要注意生徒アラート |
| S-03-02 | 学生一覧画面 | 担当クラスの生徒リスト、各生徒の進捗サマリ |
| S-03-03 | 学生詳細画面 | プロフィール詳細、就職活動履歴 |
| S-03-04 | 申請管理画面 | 学生からの申請一覧表示、承認／却下／コメント |
| S-03-05 | 求人管理画面 | 求人の一覧表示、新規作成、編集、削除、コピー |
| S-03-06 | 教員マイページ画面 | プロフィール確認／編集、ログアウト |

### 1.4 管理者画面（S-04-XX）
| 画面ID | 画面名 | 説明 |
| :--- | :--- | :--- |
| S-04-01 | 権限管理画面 | 教員アカウントの一覧、登録、編集 |
| S-04-02 | クラス管理画面 | クラスの一覧、登録、編集 |
| S-04-03 | マスタデータ管理画面 | 各種マスタの一覧、登録、編集、削除 |
| S-04-04 | 設定管理画面 | システム設定値の一覧、変更 |
| S-04-05 | システム監視画面 | システムの稼働状況を表示 |
